= Коды ошибок WEB API
:toc:
:toc-title: Оглавление

== Введение

В данном документе приведено описание кодов ошибок WEB API.

== Пример ответа для 400 http кода

[source,json]
----
[
  {
    "fieldName": null,
    "code": "UniqueTableName",
    "errorMessage": "Table with name [iris] already exists!"
  }
]
----
